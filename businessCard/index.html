Приложение для Windows

Приложение для Windows позволяет пользователям:

  • Загружать Логи боев или удалять уже существующие (личные) Логи боев;
  • Просматривать общую информацию о боях (урон, исцеление, полученный урон, ресурсы и др.);
  • Просматривать подробную информацию о боях (урон, исцеление, полученный урон, ресурсы и др.);
  • Общаться в чате с другими пользователями.

Зачем это нужно?

Details Analysis GCL позволяет анализировать бои игрокам для того, что бы:

  1. Улучшить свои показатели в бою;
  2. Узнать, какие были допущены ошибки во время боя;
  3. Улучшить или изменить тактику боя;
  4. Сравнить свой показатель боя с другими игроками и, при необходимости, изменить порядок и время использования различных талантов;
  5. Узнать, на сколько эффективна была твоя группа в бою или сравнить с другими группами;
  6. Посмотреть личные достижения в бою (сравнить с предыдущими результатами).

Как это работает?

Details Analysis GCL интерпритирует не понятный для человек Лог боя в более понятную и легкую для просмотра форму.

После интерпритации, любой пользователь, не зная каким образом создается Лог боя и какая информация записывается в Лог боя, получает наиболее важную информацию.

Пример важной информации:

  1. Информация о нанесеном уроне за весь бой или отдельно каждым игроком;
  2. Какую цель игрок атаковал в опр. промежуток времени;
  3. Когда и какие способности использовал игрок;
  4. Какой урон нанесла способность и статус способности (критический урон, промах, паррирование и др.);
  5. Сортировка информации по необходимым критериям (название способности, цель, урон и др.).

Кроме интерпритации, Details Analysis GCL позволяет сохранить данные в базу данных что бы можно было просматривать результат повторно.

Веб приложение

Веб приложение позволяет пользователям:

  • Просматривать общую информацию о боях (урон, исцеленwие, полученный урон, ресурсы и др.);
  • Просматривать подробную информацию о боях (урон, исцеление, полученный урон, ресурсы и др.);
  • Просматривать общую статистику боя;
  • Просматривать личную (каждого игрока) статистику боя;
  • Просматривать события каждого игрока во время всего боя или во ремя опр. интервала боя;

Зачем нужно Веб приложение, если есть приложение для Windows?

Есть несколько причин, почему некоторым пользователи лучше использовать Веб приложение:

  1. Веб приложение не нужно устанавливать на компьютер: достаточно зайти в Браузер и указать адрес Веб приложения;
  2. Приложение для Windows необходимо самостоятельно обновлять после выпуска каждого обновления, в отличии от Веб приложения;
  3. Веб приложение предоставляет доп. функционал в виде схем и диаграм для статистики боя;
  4. Пользователю не нужен весь функционал, предоставляемый приложением для Windows (например, сохранение Логов).
