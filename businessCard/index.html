  • Desktop app fot Winows

    The Windows app allows users to:

    • Upload Combat Logs or delete already existing (personal) Combat Logs;
    • View general information about combats (damage, healing, damage taken, resources, etc.);
    • View detailed information about combats (damage, healing, damage taken, resources, etc.);
    • Chat with other users.

  • Why it need?

    Details Analysis GCL allows players to analyze combats in order to:

    1. Improve your performance in combat;
    2. Find out what mistakes were made during the combat;
    3. Improve or change combat tactics;
    4. Compare your combat score with other players and, if necessary, change the order and timing of using different talents;
    5. Find out how effective your group was in combat or compare with other groups;
    6. View personal achievements in combat (compare with previous results).

  • How it work?

    Details Analysis GCL interprets the combat log that is incomprehensible to humans into a more understandable and easy-to-view form.

    After interpretation, any user, without knowing how the Combat Log is created and what information is written to the Battle Log, receives the most important information.

    An example of important information:

    1. Information about the damage dealt for the entire combat or separately by each player;
    2. What target the player attacked in a certain period of time;
    3. When and what abilities did the player use;
    4. What damage the ability did and the status of the ability (critical damage, miss, parry, etc.);
    5. Sorting information according to the necessary criteria (name of the ability, target, damage, etc.).

    In addition to interpretation, Details Analysis GCL allows you to save data to a database so that you can view the result again.

  • Example:

    Suppose in a game (for example, World of Warcraft) it is possible to enable the Logging of combats in especially difficult dungeons (raid dungeons).

    Combat logging will create a special file that will store information about the events that happened/are happening during the battle,

    but the information is written in a way that will be understandable to the game, but difficult to understand for the players. An example of a combat log file:

    The presented example with information about the battle is difficult for an ordinary person to understand: it is difficult to get the necessary information about the combat.

    App Details Analysis GCL reads the above information, splits the data into the required entities, extracts the information, and presents the resulting information in an understandable form.

  • Web app

    Web app allows users to:

    • View general information about combats (damage, healing, damage taken, resources, etc.);
    • View detailed information about combats (damage, healing, damage taken, resources, etc.);
    • View overall combat statistics;
    • View personal (each player) combat statistics;
    • View the events of each player during the entire battle or during the definition. combat interval;

  • Why need Web?

    There are several reasons why some users are better off using the Web Application.:

    1. The web application does not need to be installed on a computer: just go to the Browser and specify the address of the Web application;
    2. The Windows app needs to be self-updated after each update is released, unlike the Web app;
    3. The web application provides functionality in the form of diagrams and diagrams for combat statistics;
    4. The user does not need all the functionality provided by the Windows application (for example, saving Logs).
